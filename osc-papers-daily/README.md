# OSC Papers Daily

有机太阳能电池文献日报 - 每日自动抓取有机太阳能电池领域的最新学术论文。

## 功能特性

- 每日自动从 arXiv 和 Semantic Scholar 抓取最新论文
- 智能标签分类（非富勒烯受体、本体异质结、聚合物太阳能电池等）
- 支持关键词搜索和标签筛选
- 响应式设计，适配各种设备
- 数据持久化存储在 Vercel KV

## 技术栈

- Next.js 14 App Router + TypeScript
- Tailwind CSS
- Vercel KV
- Vercel Cron

## 快速开始

```bash
npm install
npm run dev
```
